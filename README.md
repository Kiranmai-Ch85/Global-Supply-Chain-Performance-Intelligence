# 🌍 Global Supply Chain Performance Intelligence

## 📌 Project Overview
This project focuses on analyzing enterprise procurement and supply chain operations to improve business decision-making using Python, SQL, and Power BI.

The project identifies supplier performance, procurement efficiency, inventory management, logistics performance, and financial insights through data analysis and interactive dashboards.

---

## 🎯 Business Objective

To build an end-to-end Supply Chain Analytics solution that helps organizations:

- Improve procurement efficiency
- Monitor supplier performance
- Analyze inventory levels
- Track logistics and delivery performance
- Support data-driven business decisions

---

## 📂 Dataset Source

**Enterprise Procurement & Vendor Performance Analytics Dataset**

The project uses multiple structured datasets covering:

- Purchases
- Sales
- Beginning Inventory
- Ending Inventory
- Purchase Prices
- Vendor Invoice
- Vendor Sales Summary
- Purchase Orders
- Procurement KPI
- Supply Chain Operations
